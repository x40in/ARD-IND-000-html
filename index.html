Ардуино и индикаторы

Цикл программ (скетчей) для Ардуино, в которых последовательно, начиная с самых простых конфигураций, иллюстрируется работа с индикаторами (светодиодами, лампочками, матрицами). Цикл начинается с управления встроенным светодиодом Ардуино, затем рассматриваются дискретные внешние светодиоды, одно- и многоразрядные светодиодные индикаторы, матрицы, системы с несколькими индикаторами. Используется Arduino UNO, подключённая своим портом USB к компьютеру под Windows, а для управления индикаторами с компьютера оператор вводит команды в мониторе последовательного порта Arduino IDE.
